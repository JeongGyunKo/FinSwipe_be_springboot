엔드포인트

GET https://api.finswipe.co.kr/news/latest
쿼리 파라미터: userId={UUID} (필수)  ·  limit=50 (기본값 20)  ·  ticker=NVDA (선택, 티커 필터)

카드 테마 결정 — sentimentLabel

📈 "positive" → 초록 (bull) 📉 "negative" → 빨강 (bear) 📊 "neutral" → 회색 (neutral) 🔀 "mixed" → 노랑 (mixed)

앞면 필드

필드타입설명
tickerstring종목 코드 (예: NVDA)
tickerNamesarray회사명 배열 [{ticker, name}] → 헤더 서브텍스트
currentPricenumber현재 주가 (달러)
changePct1dnumber전일 대비 등락률 (%) — 양수: 상승, 음수: 하락
sparklinenumber[]30일 종가 배열 → SVG 스파크라인 렌더링용
sentimentLabelstringpositive | negative | neutral | mixed → 카드 테마 결정
sentimentScorenumber감성 점수 ±100 범위, 소수점 1자리 (예: +84.0, -61.0) → big-num
sentimentReasonstring감성 판단 근거 한 줄 요약 → 점수 아래 sub 텍스트
headlineKostring한국어 헤드라인 → 하단 푸터 제목
publishedAtdatetime기사 발행 시각 (ISO 8601) → "N시간 전" 변환
eventCategorystring?이벤트 분류 태그 (예: 실적발표, 거래량 급등) → 푸터 필
